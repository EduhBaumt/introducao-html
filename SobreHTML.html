Sobre

(Voltar)

HTML (Linguagem de Marcação de HiperTexto), é o bloco de construção mais básico da web. Define o significado e a estrutura do conteúdo da web. Outras tecnologias além do HTML geralmente são usadas para descrever a aparência/apresentação (CSS) ou a funcionalidade/comportamento (JavaScript) de uma página da web.

O HTML foi inventado por Tim Berners-Lee, um físico do centro de pesquisas CERN, na Suíça. Ele surgiu com a ideia de um sistema de hipertexto na internet.

Hipertexto significa um texto que possui referências (links) para outros textos que podem ser acessados imediatamente. Ele publicou a primeira versão do HTML em 1991, consistindo em 18 tags. Desde então, cada versão do HTML vem com novas tags e atributos (modificadores de tags).

Estrutura Básica

(Voltar)

Todo documento em HTML sempre irá iniciar com essa estrutura básica, sempre será dessa forma. Programas como Sublime Text, ou o Visual Studio Code já vem com atalhos para a criação dessa estrutura, sendo assim, você não tem a "necessidade" de decorrar o código, mas é sempre bom que saiba.

Para fácil compreensão do documento HTML podemos separar em blocos as tags essencias:

  • DOCTYPE
  • HEAD
  • BODY

DOCTYPE:

O doctype não é uma tag HTML, mas uma instrução para o navegador e outros programas que podem ler seu site, que o código encontrado ali é um código HTML. Assim eles sabem o que fazer para mostrar seu site da melhor forma possível.Lembre-se: o doctype é OBRIGATÓRIO e deve ser sempre a PRIMEIRA LINHA do seu documento.

HEAD:

Contém informações que não são transpostas visivelmente para o usuário/leitor do documento. São dados implícitos, de uso e controle do documento: vinculação com outros arquivos, aplicação de lógica de programação de scripts e metadados.

BODY:

Trata-se do documento em si, tudo o que o cliente vê. É todo e qualquer texto, imagem, sons, todas mídias de saída. Além disso, toda a apresentação de entrada de dados (formulários) também se aplica neste seção do documento.

Dentro do elemento BODY sua estrutura de página terá os elementos semânticos da construção da sua página, onde serão declarados e identificados cabeçalhos, rodapé, conteúdo principal, etc.

O'que são Tags

(Voltar)

O documento HTML consiste de uma série de códigos em série, que são as tags. Elas definem a estrutura do site, podendo formas títulos para seu documento, parágrafos, e muito mais. Todas elas são compostas por uma estrutura de abertura e uma de fechamento.

Digamos que você acabou de criar o seu documento, ele aparecerá vazio, certo? Pois bem, para darmos vida ao seu "site" começamos com a tag de título "h1", escrevemos o que está na nossa mente, e terminamos o título com sua tag de fechamento "/h1".

Dessa forma, ao salvar o arquivo com a extensão .html e abri-lo em um navegador, poderemos ver o seu título no topo da página!

O uso dos Textos

(Voltar)

Os textos mais estruturados são compostos por títulos e parágrafos,esteja você lendo uma história, um jornal, um livro da faculdade, uma revista, etc. O conteúdo estruturado torna a experiência de leitura mais fácil e mais agradável.

Em HTML como vimos anteriormente temos códigos específicos para textos.

Cada título deve ser envolvido por um elemento de título, seja ela "H1", "H2", etc

Cada parágrafo deve estar envolvido por um elemente "p"

Outro elemento de texto é o "Blockquote", normalmente usado em citações à falas de alguem, ou citações de partes de livros, etc.
